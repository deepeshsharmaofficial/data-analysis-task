## Getting started

### 1. Clone the repo

```sh
git clone https://github.com/deepeshsharmaofficial/data-analysis-task
cd /data-analysis-task
```

#### 2. Install dependencies

```sh
yarn install
```

#### 3. Run the project

```sh
yarn dev
```
